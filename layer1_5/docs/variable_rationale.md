# 変数選択の根拠（野球×物理）

## 回転代理変数

### なぜ回転数"そのもの"を使わないか

- 回転数は**投球条件に依存**し、打者の制御範囲外
- 回転数と打球結果の関係は**非線形**で複雑

### 代理変数の考え方

- **回転効率**: 回転数/速度比 → 打者が「回転を活かせているか」
- **回転軸との関係**: 回転軸と打球角度の差分 → 打者の角度制御精度

## 接触質指標

### EV減衰・伸びの意味

- **理論EVと実測EVの差分**: バットの当たりの質
- **距離/EV比**: 角度と速度の最適化度合い

### 物理的解釈

- 良い当たり = EV減衰が少ない = バットの芯で捉えている
- 伸びる打球 = 角度と速度のバランスが良い

## 打撃特性

### bat_speed, swing_length

- **バットスピード**: 打球速度の主要因
- **スイング長**: スイングの効率性

### attack_angle, swing_path_tilt

- **アタックアングル**: 打球角度への直接影響
- **スイングパス**: バットとボールの出会い角度
