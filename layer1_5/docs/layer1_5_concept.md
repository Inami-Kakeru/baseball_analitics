# Layer 1.5 の概念

## なぜ1.5層が必要か

### Layer 1の限界

Layer 1では「長打が出やすい状態（EV×LA帯域）」を定義したが、これは**結果の集計**に基づく。

- 状態は定義できたが、**その状態に到達する条件**は不明
- スイング指標と状態の関係が直接見えていない

### Layer 2への橋渡し

Layer 2ではGA2M等のモデルで「yを説明する」が、**説明変数の選定**が課題。

- 全変数を投入すると過学習・解釈困難
- 物理的に意味のある変数に絞る必要がある

### Layer 1.5の役割

1. **特徴量スクリーニング**: 単変量で効果の大きい変数を特定
2. **安定性検証**: 条件を変えても効果が残る変数を選定
3. **解釈可能性確保**: 野球×物理の観点から意味のある変数に絞る

## 論文・発表での位置づけ

- **再現性の担保**: 状態定義（Layer 1）→ 特徴量選定（Layer 1.5）→ モデル構築（Layer 2）の流れを明確化
- **恣意性の排除**: 機械学習による自動選定ではなく、物理的解釈可能な変数に絞る
- **実用性**: Layer 2で使う変数を事前に確定することで、施策への落とし込みを容易に
