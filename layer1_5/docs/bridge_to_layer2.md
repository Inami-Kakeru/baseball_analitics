# Layer 2への橋渡し設計思想

## ここからどう施策に落とすか

### Layer 1.5の出力

- **候補特徴量リスト**: 効果量・安定性・解釈可能性が評価済み
- **特徴量グループ**: 物理特性・打撃特性・投球条件の分類

### Layer 2での活用

1. **特徴量選定**: Layer 1.5の候補から最終選定
2. **モデル構築**: GA2M等で非線形関係を捉える
3. **解釈**: 各特徴量の効果を可視化

### 施策への落とし込み

#### 打撃指導への応用

- **効果の大きい特徴量** → 重点的に改善すべきスイング要素
- **安定性の高い特徴量** → 再現性の高い指導ポイント

#### データドリブンな改善

- Layer 1.5で「何が効くか」を特定
- Layer 2で「どの程度効くか」を定量化
- 施策: 「その要素をどう改善するか」を設計

## 設計原則

1. **解釈可能性優先**: ブラックボックスな変数は避ける
2. **物理的妥当性**: 野球×物理の観点から意味がある変数に絞る
3. **再現性**: 条件を変えても効果が残る変数を選ぶ
4. **実用性**: 施策に落とし込みやすい変数を優先
