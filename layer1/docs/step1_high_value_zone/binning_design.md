# EV×LA ビン設計

## ビン幅

- Exit Velocity（EV）：2 mph
- Launch Angle（LA）：2 deg

## 理由

- ビンが粗すぎる場合：
  - 高価値帯の形状が潰れる
- ビンが細かすぎる場合：
  - サンプル数不足でノイズが支配的になる

2mph × 2deg は、

- Statcast実務分析
- 先行研究
- 本データ量（年2000〜3000BIP）

を踏まえた **実務的最適解**。

## 各セルで計算する値

- count：サンプル数
- E[V]：長打スコア V の平均

count < N_min（例：20 or 30）のセルは

- 不安定とみなし除外（NaN）
