# 高価値帯（High-Value Zone）という考え方

本研究では「長打力」を以下のように再定義する。

- 長打 = 結果（HR, 2B, 3B）そのものではない
- 長打 = **再現可能な物理状態に入った打球の集合**

従来指標の問題点：

- Barrel：
  - MLB全体平均に基づく定義
  - 選手固有性を捉えにくい
- xwOBA：
  - EV・LAを説明変数としてMLで構築
  - 今回の分析で再利用すると循環論法になる

そこで本研究では、

> 「EV×LA 空間上で、平均的に"長打価値"を生みやすい領域」

を **High-Value Zone（高価値帯）** と定義し、
これを選手固有に算出する。

この高価値帯は、

- 打球結果ではなく
- 物理状態としての「成功ゾーン」

を表すものであり、
「安定性を上げる」という最終目的に直結する。
