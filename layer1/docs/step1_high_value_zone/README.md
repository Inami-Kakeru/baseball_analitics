# Step1: High-Value Zone（高価値帯）定義

本ドキュメント群は、本研究における **Step1（第1層）** の設計思想・数理定義・実装仕様をまとめたものである。

Step1の目的は、

- 「結果（HR, xwOBA）」ではなく
- **打球の物理状態（EV×LA）に基づいて**
- 「再現可能な高価値打球の空間（High-Value Zone）」を定義すること

である。

本Stepで定義された高価値帯およびラベル y₁ は、

- Layer1.5（物理特性の一次スクリーニング）
- Layer2（安定性向上のための施策提案）

の **不変な前提条件（アンカー）** として扱われ、後段で再定義されることはない。
